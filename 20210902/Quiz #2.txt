Quiz #2

Objetivo: Crear manualmente múltiples documentos HTML y modificar algunos valores de 
presentación del documento utilizando CSS.

- Tomar como base el documento HTML creado en el Quiz #1.

1. Crear un nuevo documento HTML para contener el contenido de cada título (sección) del documento del Quiz #1. Valor: 2,5 puntos.
2. Sustituir el lugar en donde se ubicaba el contenido trasladado a un documento HTML independiente, por un enlance <a>.  Valor: 2,5 puntos.
3. Crear un nuevo archivo CSS con el nombre main.css en la carpeta de nombre css. Trasladar las reglas de CSS que ya utilizaba
   en el documento original y coloquelas en el archivo main.css.  Valor: 2,5 puntos.
4. Modifique todos los documentos HTML generados para que hagan referencia al archivo de CSS main.css.  Valor: 2,5 puntos.

Opcional: Publicar en su GitHub Page.

Ejemplo de posible estructura resultante:

	- index.html
	- seccion1.html
	- seccion2.html
	- seccion3.html
	- seccion4.html
	- css/main.css
	
Ejemplo de posible estructura de index.html:

	<!DOCTYPE html>
	<html lang="en">

	<head>
		<meta charset="UTF-8">
		<meta http-equiv="X-UA-Compatible" content="IE=edge">
		<meta name="viewport" content="width=device-width, initial-scale=1.0">
		<title>Organización territorial de Costa Rica</title>
	</head>

	<body>
		<header>
			<h1>Organización territorial de Costa Rica</h1>
		</header>
		<main>
			<section>
				<p>Al 2021 la organización territorial comprende de 7 provincias, ..
				</p>
			</section>
			<ul>
				<li><a href="historia.html">Historia</a></li>
				<li><a href="provincial.html">Organización provincial</a></li>
			</ul>
		</main>
	</body>

	</html>
