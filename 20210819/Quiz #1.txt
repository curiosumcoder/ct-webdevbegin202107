Quiz #1

Objetivo: Crear manualmente un documento HTML y modificar algunos valores de presentación del documento utilizando CSS.

- HTML
	- Haga uso de al menos de los elementos: h1, h2, section, ul, li, figure, figcaption, img, p, table, thead, tbody, tr, td. Valor: 3 puntos
	- Se deben incluir al mentos 3 títulos o subtítulos. Valor: 1 punto.
	- Se deben incluir al menos 3 párrafos de texto. No hay límite definido para la cantidad de texto. Valor: 1 punto.
	
- CSS
	- Utilizando CSS, modificar el font-family del documento. Cambiar a un tipo de letra de su gusto. Valor: 1 punto.
	- Utilizando CSS, agregar un borde superior a todos los títulos o subtítulos. Usted define el tamaño, color y ancho. Valor: 2 puntos.
	- Utilizando CSS, modifique al menos dos atributos de presentación de algunos elementos de su elección dentro del documento. Valor: 2 puntos.
